# Politique de sécurité

## Versions supportées

Le projet suit le versionnage sémantique. Seule la dernière version mineure
publiée reçoit les correctifs de sécurité.

| Version       | Supportée |
| ------------- | --------- |
| Dernière `0.x` | ✅        |
| Versions antérieures | ❌  |

## Signaler une vulnérabilité

**Ne pas ouvrir d'issue publique pour une faille de sécurité.**

Privilégiez les **GitHub Security Advisories** (onglet *Security* →
*Report a vulnerability*) du dépôt, ou à défaut un e-mail à
<statoon54@gmail.com>.

Merci d'inclure :

- une description de la vulnérabilité et de son impact,
- les étapes de reproduction (ou un PoC),
- la version concernée (`mailhive version`).

Vous recevrez un accusé de réception sous quelques jours ouvrés. Merci de
laisser un délai raisonnable pour le correctif avant toute divulgation publique.
